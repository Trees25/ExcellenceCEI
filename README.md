# Excellence CEI

Landing page del Centro de Estética Integral Excellence.

## Stack
- Vite + React (JSX)
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Uso

```bash
npm install
npm run dev
```

Build de producción:
```bash
npm run build
```

## Estructura
- `src/App.jsx` — punto de entrada de la página
- `src/components/SiteHeader.jsx` — navegación
- `src/components/sections/*` — secciones (Hero, About, Services, Team, Academy, Testimonials, Contact, Footer)
- `src/assets/*.jpg` — imágenes (reemplazá por las reales)
- `src/styles.css` — tokens de diseño + Tailwind
